Ejercicio #4: EDAD PROMEDIO CON WHILE

Se requiere un algoritmo para obtener la edad promedio de un grupo de 'N' alumnos.
Realice el diagrama de flujo Y el pseudocódigo para representarlo, utilizando los tres tipos de estructuras de ciclo

Pseudocodigo

Variable Descripción Tipo
N Numero de alumnos que se promediará la edad Int
p 0, inicilización del ciclo de suma Int
i iteración del ciclo de sumatoria = 1 Int
promedio promedio de edades Int / Float
numero edades de los alumnos Int / Float
  1. Inicio
  2. Leer N, p , i, numero
  3. comprobar si i <= N
  4. si lo es; sumar las edades correspondientes numero + numero...
  5. Repetir el paso anterior mientras cumpla la condición del paso 3
  6. Sacar el promedio de las edades, p / N = promedio
  7. Mostrar promedio
  8. FIN
