Ejercicio #1: SUMA 10 NUMEROS CICLO WHILE

Se requiere un algoritmo para obtener la suma de diez cantidades mediante la utilización de un ciclo “WHILE”.
Exprese el algoritmo usando Pseudocódigo y diagrama de flujos

Pseudocodigo

Variable Descripción Tipo
n Variable a sumar = 0 Int
i Ciclo de itinerancia = 0 ( hasta el 10) Int
numero número ingresado a sumar Int / Float
  1. Inicio
  2. Leer n, i ; ingresar numero
  3. Si i <= 10, Sumar numero + n = n ; i = i+1
  4. Repetir ciclo si i <= 10
  5. Mostrar R
  6. Fin
