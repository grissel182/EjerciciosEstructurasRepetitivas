Ejercicio #7: TABLAS DE MULTIPLICAR

Construir la tabla de multiplicar de un número ingresado por teclado, partiendo desde cero.
Exprese el algoritmo usando Pseudocódigo y diagrama de flujos

Pseudocodigo

Variable Descripción Tipo
n número que creará la tabla de multiplicar Int / Float
i Número de ciclos a multiplicar Int / Float
R resultado de la multiplicación = i * n Int
  1. Inicio
  2. Leer n , i
  3. i <= 10 , i = i + 1 , n * 1 = r
  4. Mostrar r
  5. Repetir punto 3 hasta que i =10
  6. Mostrar r
  7. Fin
